# Contributing to Sovereign AI Scanner

We welcome contributions! As a Sovereign AI project, we value:
1. **Privacy**: No features that leak data to central servers.
2. **Speed**: Lightweight scripts over heavy frameworks.
3. **Sovereignty**: Tools that empower the user.

## How to contribute
1. Fork the repo.
2. Create a branch.
3. Submit a PR.
